# 🌊 Wave-Haven
> 浪潮不息，归藏有道 | Wave Flows, Knowledge Rests

Wave-Haven 是一套**AI原生的多Agent协同工作系统**，定位为个人/团队的"数字工作中枢"，将任务调度、知识管理、多Agent协作能力融为一体，完全替代传统的项目管理+知识库+TODO工具组合。

---

## ✨ 核心特性
### 🌊 Wave 任务流引擎
- **智能任务编排**：一句话输入自动拆解为多Agent协作的任务流
- **子任务依赖**：支持复杂工作流的依赖管理和自动调度
- **实时进度追踪**：自动计算任务完成率，状态实时更新
- **AI规划器**：自动分配Agent、估算时间、优化执行路径

### 🏝️ Haven 知识沉淀引擎
- **双轨记忆架构**：归藏-标签（永久知识）+ 归藏-时序（生命周期记忆）
- **四层时序存储**：short(7天) → mid(30天) → long(90天) → dream(永久灵感)
- **本地语义搜索**：基于Qwen3嵌入模型，无需联网即可搜索所有历史知识
- **自动知识提取**：任务完成后自动沉淀经验到知识库，实现"做过即存档"

### 🤖 多Agent协同体系
| Agent | 角色 | 核心能力 |
|-------|------|----------|
| **Nova** | 浪潮指挥官 | 任务调度、全局协调 |
| **Luna** | 归藏守护者 | 知识管理、记忆检索 |
| **Kiki** | 优化工程师 | 代码优化、架构改进 |
| **Coco** | 执行专家 | 脚本运行、任务执行 |
| **DreamNova** | 梦境推演师 | 方案验证、架构决策 |

### 🧠 AI原生能力
- **全本地化运行**：所有AI模型和数据都在本地，隐私安全，无API依赖
- **智能路由**：自动识别任务类型，分配给最合适的Agent处理
- **自动演进**：定期分析业务需求，自动推荐和部署新的专用Agent
- **语义理解**：自然语言交互，无需学习复杂命令

## 🚀 快速开始
### 环境要求
- Python 3.8+
- Git
- （可选）Ollama + Qwen3 系列模型（用于本地语义搜索和AI能力）

### 安装部署
```bash
# 1. 克隆项目
git clone https://github.com/sussywafula00-bit/wave-haven.git
cd wave-haven

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入所需配置

# 4. 初始化系统
./wave-haven init

# 5. 启动系统
./wave-haven start
```

### 基础使用
```bash
# 创建一个新浪潮
./wave-haven wave create "开发用户认证系统"

# 查看所有浪潮
./wave-haven wave list

# 搜索知识库
./wave-haven haven search "用户认证"

# 查看系统状态
./wave-haven status
```

## 🛠️ 技术架构
```
┌─────────────────────────────────────────────────┐
│                  前端交互层                      │
│  CLI工具  /  Web UI  /  自然语言接口              │
├─────────────────────────────────────────────────┤
│                  核心引擎层                      │
│  🌊 Wave 调度器  /  🏝️ Haven 知识库  /  🤖 Agent路由 │
├─────────────────────────────────────────────────┤
│                  AI能力层                       │
│  本地LLM  /  嵌入模型  /  语义索引  /  规划器     │
├─────────────────────────────────────────────────┤
│                  存储层                         │
│  SQLite  /  文件系统  /  向量索引                │
└─────────────────────────────────────────────────┘
```

### 核心技术栈
- **后端**: Python 3.8+
- **Web框架**: FastAPI + Streamlit
- **AI模型**: Qwen3 (本地部署)
- **存储**: SQLite + 文件系统 + FAISS向量索引
- **部署**: 单机部署，无外部依赖
