# Wave-Haven

**Multi-Agent Task Orchestration & Knowledge Management System**

<p align="center">
<strong>"Wave flows, Haven shelters"</strong><br>
<strong>"浪潮不息，归藏有道"</strong>
</p>

## Overview

Wave-Haven combines:
- **Wave System**: Task orchestration and multi-agent coordination
- **Haven System**: Knowledge management with 4-layer memory architecture

## Installation

```bash
git clone https://github.com/sussywafula00-bit/wave-haven.git
cd wave-haven
./install.sh
```

## Quick Start

```bash
# Start the system
claw skills run wave-haven --start

# Create a wave (task)
claw skills run wave-haven --create-wave "Analyze data"

# Query knowledge
claw skills run wave-haven --query "project info"
```
